Before you start

I made this guide as an easy and convenient reference on my quest to play all the Sonic games (or at least most of them).
Below, you'll find a list of a bunch of sonic games and what I believe to be the definitive version to play them. This list wont be perfect. I might miss stuff, or new mods or game versions may be released.
If you find an error, or a section that fails to mention an alternative way to play the game, please message me on Bluesky so I can edit this page.

The legality of emulation

This is a really difficult topic to discuss, since there are no consistent court rulings to go off of.
If you want a heavily summarized and cut down explanation:
-Downloading an Emulator is always fine. They don’t contain copyrighted material.
-Extracting the ROM of a game you physically own and emulating that shouldn’t get you into trouble. You own that game.
-Downloading ROM’s from the internet is a different story. I don’t recommend you do this, especially for games that are still actively sold.
Usually, only the people distributing the ROM’s get into trouble, but if you expect SEGA employees to knock at your door tomorrow
to search your PC for ROM’s, you should probably stay away from online downloads.

3D Games

Sonic Adventure

Vanilla Sonic Adventures definitive release is its Dreamcast release. Every version after this one has only downgraded the game
(In both objective and subjective ways). Instead of emulating the game, we'll use its PC release as a basis,
bring it as close to the Dreamcast version as possible, and add a few small quality of life mods.
We’ll use the SADX Mod Installer for this, since it comes prepackaged with everything well need,
and also turns the 2011 Steam release into the 2004 PC version. Most mods are made for the 2004 version.

Definitive Version:

-Sonic Adventure DX (Steam / 2004 PC) + SADX Mod Installer

Alternatives:

-Sonic Adventure (Dreamcast)

Guide:

(If you use the non-Steam version as a basis, the installer might not work. I havent tested it yet. IF this should be the case,
just download the Mod Manager directly and search for the mods you want to add.)

The Installer comes prepackaged with the Mod Manager, and all of the mods we'll need. Select Custom Mode,
make sure all the useful options like auto updates are turned on, and select the mods you want.
Since we want to active most of them, i've just listed the ones you might NOT want to use:
-Dreamcast Character Pack (Only contains the Dreamcast models. The Textures are part of another mod.
Avoid this if you like the lankier DX models better, add it if you prefer the Dreamcast models)
-Pause Hide (You don't need this. Its Part of another mod)
-Steam Achievements (Remove if you don't own the Steam version, otherwise leave it in)
-ADX Voices and Music (Remove if you own the Steam version. It already comes with the better audio)
-Super Sonic (in action Stages) (Remove if you don’t want super sonic as a post story unlock)
Again, you can just activate everything else.
Tip: “SADX Style Water” mods in the original water. Its just named weirdly.

Now start the Mod Manager (Not the Game) and Launch the game via the manager. Have fun playing ;3
